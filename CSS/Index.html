Usando flexbox pra centralizar

Aqui foi usado a seguinte estrutura: No elemento pai, foram utilizadas as regras:

  • display: flex;
  • flex-direction:column;
  • justify-content: center;
  • align-items: center;
  • justify-content: space-around;
  • margin: 0 auto;// Utilizado para centralizar o pai verticalmente

Utilizando essas regras, os elementos filhos (as 3 caixas) ficaram centralizados em forma de coluna horizontalmente e verticalmente e ficaram com espaços iguais distribuidos

Aqui foi usado a seguinte estrutura: No elemento pai, foram utilizadas as regras:

  • display: flex;
  • flex-direction:row;
  • justify-content: center;
  • align-items: center;
  • justify-content: space-around;
  • margin: 0 auto;// Utilizado para centralizar o pai verticalmente

Utilizando essas regras, os elementos filhos (as 3 caixas) ficaram centralizados em forma de linha horizontalmente e verticalmente e ficaram com espaços iguais distribuidos
